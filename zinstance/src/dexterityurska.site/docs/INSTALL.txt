dexterityurska.site Installation
--------------------------------

To install dexterityurska.site using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``dexterityurska.site`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        dexterityurska.site
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
